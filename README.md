## 프로젝트 개요 
애플리케이션 사용자의 리뷰 데이터를 받아 기업의 분석 및 개발에 중요한 자료로 활용할 수 있는 웹 서비스입니다.

<table align="center">
  <tr>
    <th>프로젝트 기간</th>
    <td>2022.01 ~ 2022.04 (3개월)</td>
  </tr>
  <tr>
    <th>목표 설정</th>
    <td>어플 리뷰에 대한 객관적 분석 가능</td>
  </tr>
  <tr>
    <th>기대 효과</th>
    <td>
      1. 사용자가 마켓에 대해 어떠한 생각을 가지고 있는지 알 수 있음 </br>
      2. 피드백을 통한 개선점을 파악 가능 </br>
      3. 마케팅 부서에 중요한 자원
    </td>
  </tr>
</table>
